独立的一个渲染区域

块级 display:block|table|list-item

行级 display:inline|inline-block|inline-table

BFC默认规则

  1. 默认，内部块元素从上到下放置，独占一行
  2. 渲染区域相互隔离，及外部不影响内部，内部不影响外部，特殊方式除外
  3. margin bottom和top 会出现合并现象
  4. 浮动元素不会对BFC的区域造成影响

形成条件

  1. float的值不为none
  2. position的值不为static或relative
  3. overflow不为visible
  4. display为flex|inline-flex|inline-block|table-cell|table-caption

解决的问题

  1. 垂直方向margin合并 是要合并的margin-top的元素用BFC盒子包裹
  2. 垂直方向margin溢出

      解决方法:

    • 将父元素设置成BFC
    • 父元素添加上边框，添加透明色，设置盒模型
    • 将margin-top换成padding-top
    • 父元素内添加一个table标签，即添加一个BFC
    • 父元素添加伪元素::before{ content:""; display:table; }
  3. 自适应两栏 左侧定宽+浮动，右侧overflow:hidden
  4. 防止高度塌陷
